## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Customization

### Colors

The color scheme can be customized in the `tailwind.config.js` file. The primary color is defined in the theme configuration.

### Content

To update the content:
1. Update the data in the respective section components
2. Replace images in the `public` directory
3. Update social media links and contact information

## Deployment

The project can be deployed to any static hosting service. Here's how to build for production:

```bash
npm run build
# or
yarn build
```

The build files will be generated in the `dist` directory.
